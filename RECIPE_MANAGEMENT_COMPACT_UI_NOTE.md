# 레시피 관리 화면 간소화

## 변경 사항

- 상단의 `내 레시피 보관함` 제목과 설명 영역을 제거했습니다.
- 상단 탭을 `내 레시피`, `저장한 레시피` 버튼으로 단순화했습니다.
- 각 버튼에는 현재 항목 수를 작은 배지로 표시합니다.
- 레시피 카드에서 온도 정보를 제거했습니다.
- 레시피 설명을 목록에서 제거하고 제목을 한 줄로 표시합니다.
- 썸네일, 카드 안쪽 여백, 카드 사이 간격, 메뉴 버튼 크기를 줄였습니다.
- 조리 시간과 제작자 이름을 같은 줄에 배치하여 카드의 세로 높이를 줄였습니다.
- 카드를 누르면 기존과 동일하게 레시피 상세 화면으로 이동합니다.
