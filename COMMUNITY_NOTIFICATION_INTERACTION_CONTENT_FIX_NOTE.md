# 커뮤니티 알림 내용 표시 수정

- 게시글에 댓글이 달리면 알림 두 번째 줄에 새로 작성된 댓글 내용을 표시합니다.
- 댓글에 답글이 달리면 알림 두 번째 줄에 새로 작성된 답글 내용을 표시합니다.
- 게시글 제목이나 답글 대상 원댓글은 더 이상 새 알림의 두 번째 줄에 표시하지 않습니다.
- 새 댓글/답글 ID를 알림과 연결하여 이후 내용이 수정된 경우 현재 내용을 조회합니다.
- 기존 SQLite DB는 서버 시작 시 `target_reply_id` 열이 자동 추가됩니다.
- 과거 알림은 작성자와 작성 시각을 기준으로 가능한 경우 실제 댓글/답글 내용을 복구합니다.
