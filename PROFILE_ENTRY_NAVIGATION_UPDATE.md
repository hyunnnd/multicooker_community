# 프로필 진입 구조 변경

- 홈 우측 상단의 U 원형 아이콘을 현재 사용자 프로필 이미지/닉네임 이니셜로 변경했습니다.
- 홈 프로필 아이콘을 누르면 현재 사용자의 작성자 프로필로 이동합니다.
- 마이페이지 프로필 영역의 연필 아이콘을 제거하고 `프로필 보기 >`로 변경했습니다.
- 마이페이지 프로필 영역을 누르면 현재 사용자의 작성자 프로필로 이동합니다.
- 현재 사용자의 작성자 프로필에만 연필 아이콘을 표시합니다.
- 연필 아이콘에서 프로필 사진과 닉네임을 변경할 수 있습니다.
- 변경 후 홈, 마이페이지, 게시글, 댓글, 답글, 후기 및 작성자 프로필에 변경 내용을 다시 동기화합니다.
- DB 파일, 업로드 파일, 로그 및 .env는 패키지에 포함하지 않습니다.
