# 선택 병합 수정 사항

- 앱마스터의 마이페이지(SettingsScreen) 화면을 원본 형태로 복구했습니다.
- 앱마스터의 레시피 관리, 저장한 레시피, 내가 쓴 후기, 내가 쓴 댓글, 조리 이력 화면 UI를 유지했습니다.
- 두 번째 프로젝트에서는 커뮤니티 기능과 앱 설정(AppSettingsScreen), 차단 사용자 관리 기능만 유지했습니다.
- 기기 관리와 튜토리얼 다시 보기는 앱마스터 구현을 그대로 사용합니다.
- 앱 설정 및 차단 사용자 화면 상단은 기기 관리와 동일한 회색 배경, 왼쪽 뒤로가기, 오른쪽 작은 회색 제목을 사용합니다.

## 원인

이전 병합에서 `settings_screen.dart` 전체를 교체하면서 설정 하위 화면뿐 아니라 마이페이지 루트 화면까지 두 번째 프로젝트의 UI로 바뀌었습니다. 이번 수정에서는 마이페이지 루트와 관련 메뉴를 앱마스터에서 복구하고, 설정 하위 화면만 선택적으로 결합했습니다.
