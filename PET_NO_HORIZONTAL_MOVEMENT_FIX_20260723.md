# 펫 자동 좌우 움직임 및 인접 이미지 노출 수정

- `tangerine_chef.webp`, `tangerine_idle.png` 두 에셋만 사용합니다.
- 미연결 상태의 `tangerine_idle.png`는 가로 프레임을 순차 이동하지 않고 첫 프레임의 안전 영역만 사용합니다.
- 첫 프레임 좌우 12px을 제외하여 셀 경계의 잔여 픽셀이 보이지 않게 했습니다.
- 자동 x축 이동, x축 확대/축소, 회전을 제거했습니다.
- 대기 표현은 y축 방향의 미세한 숨쉬기 효과만 적용합니다.
- 사용자가 직접 드래그하여 펫 위치를 옮기는 기능은 유지합니다.
- 블루투스 연결 성공 후 홈 이동 수정도 기존 전체 프로젝트에 포함되어 있습니다.
