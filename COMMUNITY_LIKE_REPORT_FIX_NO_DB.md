# 커뮤니티 좋아요/신고 관리 수정

- 게시글 좋아요, 댓글, 답글, 관리자 인기도 변경은 게시글의 `수정됨` 표시를 만들지 않습니다.
- 게시글 제목/본문/이미지를 실제로 수정한 경우에만 `content_updated_at`이 기록됩니다.
- 기존 SQLite DB는 삭제하거나 교체하지 않습니다. 서버 시작 시 필요한 컬럼만 안전하게 추가합니다.
- 관리자 신고 관리의 처리 버튼은 `반려`, `삭제` 두 개만 표시합니다.
- 신고 처리 후 Provider 변경을 즉시 구독하여 목록과 통계가 바로 갱신됩니다.
- 배포 ZIP에는 `multicooker.db`, `*.db`, `logs/`, `local_uploads/`, `.env`가 포함되지 않습니다.
