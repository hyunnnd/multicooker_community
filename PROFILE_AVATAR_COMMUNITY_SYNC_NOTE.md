# 프로필 사진 커뮤니티 전역 반영

- 커뮤니티 게시글, 댓글, 답글, 레시피 후기, 레시피 댓글 모델에 `avatarImageUrl`을 추가했습니다.
- 서버의 `avatar_image_url` 응답을 Flutter 모델에서 파싱하도록 연결했습니다.
- 홈의 커뮤니티 최신글, 커뮤니티 목록/상세, 댓글/답글, 후기, 알림에서 실제 프로필 사진을 표시합니다.
- 사진이 없거나 이미지 로딩이 실패하면 기존 프로필 색상과 닉네임 첫 글자를 표시합니다.
- 상대경로 이미지 URL도 `API_BASE_URL` 기준 절대경로로 변환합니다.
- 알림도 작성자의 최신 닉네임, 색상, 프로필 사진을 사용자 테이블 기준으로 반환합니다.
- 프로필 수정 성공 후 기존 `CommunityProvider.load(silent: true)`가 목록을 다시 불러오므로 앱 재시작 없이 반영됩니다.
- `CommunityProvider.applyCurrentUserProfile()`가 현재 메모리의 게시글·댓글·답글·후기·레시피 댓글을 즉시 갱신하고, 이후 서버 재조회로 최종 동기화합니다.
