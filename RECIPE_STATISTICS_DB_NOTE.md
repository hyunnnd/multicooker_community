# 레시피 카드 통계 DB 연동

레시피 목록 카드에 표시되는 임시 숫자를 제거하고 로컬 FastAPI DB 값을 사용하도록 변경했습니다.

- 별점: `recipe_reviews.rating`의 평균
- 별점 옆 괄호: 삭제되지 않은 후기 개수
- 불꽃 아이콘 숫자: 완료 상태의 `cooking_history` 사용 횟수

레시피 API 응답에 아래 필드가 추가됩니다.

```json
{
  "rating_average": 4.5,
  "review_count": 12,
  "usage_count": 37
}
```

후기가 없으면 `0.0 (0)`, 사용 기록이 없으면 `0`으로 표시됩니다.
사용 횟수는 조리 완료 이력이 생성될 때 증가합니다.
