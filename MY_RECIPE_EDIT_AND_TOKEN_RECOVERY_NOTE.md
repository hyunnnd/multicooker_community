# 내 레시피 수정 및 개인 API 토큰 복구

## 내 레시피 수정

- 마이페이지 > 내가 올린 레시피 > 메뉴 > 수정에서 기존 내용을 불러옵니다.
- 레시피 상세 화면에서도 본인 레시피에만 수정 아이콘이 표시됩니다.
- 제목, 소개, 재료, 조리 단계, 온도와 시간을 수정할 수 있습니다.
- 개인 FastAPI는 `PATCH /users/me/recipes/{recipe_id}`를 사용합니다.
- 서버는 bearer token의 사용자와 레시피 소유자가 일치하는 경우에만 수정하며, 다른 계정은 403으로 차단합니다.

## 앱 재실행 후 개인 API 토큰 복구

- 회사 로그인 토큰과 개인 API 토큰을 별도로 관리합니다.
- 개인 API 토큰에 연결된 이메일도 보안 저장소에 함께 기록합니다.
- 앱 시작 시 회사 계정을 확인하고 동일 계정의 개인 API 세션을 검증합니다.
- 개인 access/refresh token이 없거나 개인 서버 재시작으로 무효화된 경우 회사 계정 정보로 `/auth/local_sync`를 다시 호출합니다.
- 커뮤니티·마이페이지 요청이 401을 받더라도 개인 API 세션을 자동 재생성하고 원래 요청을 한 번 재시도합니다.
- 일시적인 동기화 실패가 발생해도 기존의 동일 계정 개인 토큰을 성공적인 교체 전까지 먼저 삭제하지 않습니다.
- 마이페이지에는 개인 API를 수동으로 다시 연결할 수 있는 안내가 표시됩니다.

## 적용

`main.py`에 레시피 수정 API가 추가되었으므로 서버 파일을 교체한 뒤 재시작해야 합니다.

```powershell
python -m uvicorn main:app --host 0.0.0.0 --port 8001 --reload
```

기존 데이터 유지를 위해 다음 항목은 덮어쓰거나 삭제하지 않습니다.

```text
multicooker.db
local_uploads/
.env
```
