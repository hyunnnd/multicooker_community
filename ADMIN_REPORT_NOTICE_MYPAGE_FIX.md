# 마이페이지·신고 관리·공지 상세 수정

## 마이페이지
- 마이페이지 메뉴에서 `내가 쓴 댓글` 항목을 제거했습니다.
- 기존 댓글 API와 화면 코드는 다른 기능과의 호환을 위해 유지하지만 마이페이지에서는 진입하지 않습니다.

## 관리자 신고 관리
- 신고 반려 전용 API를 추가했습니다.
  - `POST /admin/community/reports/{report_id}/reject`
- 신고 콘텐츠 삭제 전용 API를 추가했습니다.
  - `POST /admin/community/reports/{report_id}/delete-content`
- 게시글 삭제 시 소속 댓글과 답글도 함께 숨김 처리합니다.
- 댓글 삭제 시 소속 답글도 함께 숨김 처리합니다.
- 동일 콘텐츠에 들어온 다른 미처리 신고는 콘텐츠 삭제 시 함께 처리 완료됩니다.
- 처리 중 중복 탭을 막고 처리 성공·실패 결과를 SnackBar로 표시합니다.

## 공지 상세
- 상단 우측 `공지사항` 문구를 15px 회색 글씨로 축소했습니다.
- 실제 공지 제목과 본문 정렬은 변경하지 않았습니다.
- 공지 목록의 이전·다음 버튼을 다시 중앙에 배치하고 전체보기는 우측에 배치했습니다.
