# 마이페이지 사용자 개발본 복구

이번 수정은 appmaster를 전체 기준으로 유지하면서, 사용자가 직접 개발한 두 번째 프로젝트의 마이페이지 영역을 다시 적용한 작업입니다.

## 사용자 개발본으로 적용한 영역
- 마이페이지 프로필 영역
- 레시피 관리
- 레시피 관리 내부의 내 레시피 / 저장한 레시피 탭
- 저장한 레시피 화면
- 내가 쓴 후기
- 내가 쓴 댓글
- 조리 이력

## appmaster 원본을 유지한 영역
- 기기 관리 화면
- 홈 스포트라이트 방식의 튜토리얼 다시 보기
- 홈, 조리, AI 및 그 밖의 appmaster 기능

## 상단 화면 형식
마이페이지 하위 화면은 `SectionPageAppBar`를 사용하여 기기 관리 화면과 같은 회색 배경, 동일한 뒤로가기 버튼, 오른쪽의 작은 회색 제목을 사용합니다.

## 인증 서버 기본 주소
`https://api.graphenesq.co.kr`로 변경했습니다.
