# 커뮤니티 스타일 및 Google 로그인 표시 방식 수정

## 커뮤니티 목록

- 전체 배경을 `#F8FAFC`로 통일했습니다.
- 게시글과 공지 카드는 흰색 또는 연한 오렌지 배경, `#E5E7EB` 계열 1px 테두리, 16px 모서리를 사용합니다.
- 카드 사이 간격을 12px, 좌우 여백을 16px로 통일했습니다.
- 커뮤니티 제목을 왼쪽 정렬하고 20px 굵은 글자로 변경했습니다.
- 관리자, 알림, 검색 아이콘을 홈 화면과 유사한 회색 아이콘 박스 안에 배치했습니다.
- 알림 개수 배지는 아이콘 박스 우측 위에 유지했습니다.
- 검색 입력 영역은 회색 배경과 테두리를 사용합니다.
- 상단 고정 공지의 `공지사항`과 제목을 오른쪽 정렬했습니다.
- 카드 그림자를 제거하고 테두리 중심으로 정리했습니다.

## 내가 쓴 후기

- 작은 화면에서 발생하던 `RenderFlex overflowed by 6.5 pixels` 가능성을 줄였습니다.
- 별 5개 표시를 `별 아이콘 + 평점`의 작은 배지로 변경했습니다.
- 날짜 영역에 `Expanded`와 말줄임 처리를 적용했습니다.
- 카드 전체가 레시피 이동 역할을 하므로 우측 화살표를 제거했습니다.

## Google 로그인

- 일반 Chrome 앱으로 완전히 전환하던 `LaunchMode.externalApplication` 대신, 지원되는 기기에서는 `LaunchMode.inAppBrowserView`를 사용합니다.
- Android에서는 Chrome Custom Tabs 계열, iOS에서는 시스템 인앱 브라우저 화면을 통해 로그인 페이지를 표시합니다.
- 인앱 브라우저를 지원하지 않는 환경에서는 기존 외부 브라우저 방식으로 자동 대체합니다.
- Android 11 이상에서 Custom Tabs 지원 여부를 확인할 수 있도록 Manifest의 `<queries>`를 추가했습니다.

## 실행

Provider와 네이티브 Manifest 변경이 포함되어 있으므로 Hot Reload 대신 앱을 완전히 종료한 후 실행합니다.

```bash
flutter clean
flutter pub get
flutter run
```
