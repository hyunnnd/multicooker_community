# 커뮤니티 댓글 입력/프로필/게시글 등록 수정

- 댓글 및 답글 입력창의 텍스트를 세로 중앙 정렬했습니다.
- 댓글 입력창 왼쪽 프로필을 현재 로그인 사용자 프로필(닉네임/아바타 색상)과 동일하게 표시합니다.
- 커뮤니티 진입 시 로컬 프로필 요약을 갱신하여 서버가 댓글에 저장하는 아바타와 화면 표시가 일치하도록 했습니다.
- 게시글 등록 성공 여부를 확인한 뒤 성공한 경우에만 목록으로 이동하고, 실패하면 오류 메시지를 화면에 표시합니다.
- 북마크 API는 제거된 상태를 유지하되, 이전 DB의 `communitypost.bookmarks NOT NULL` 열 때문에 신규 게시글 INSERT가 실패하는 문제를 위해 내부 호환 필드와 자동 스키마 보정을 추가했습니다.
