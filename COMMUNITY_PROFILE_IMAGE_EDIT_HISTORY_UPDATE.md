# 커뮤니티 작성자·알림 이동·이미지·수정 이력 업데이트

## 적용 기능

- 알림에서 게시글 상세 진입 시 대상 댓글 또는 답글 위치로 자동 이동
- 대상 댓글과 답글을 3초 동안 강조 표시
- 게시글·댓글·답글·알림의 작성자 영역을 눌러 작성자 프로필 진입
- 작성자 프로필에서 작성 게시글과 공개 레시피 조회
- 작성자/관리자 표시 문구 통일
- 게시글 및 후기 이미지를 전체 화면에서 확대·축소하고 좌우 이동
- 게시글·댓글·답글·후기·공지사항에 수정됨 표시
- 설정 모델을 알림, 앱 동작, 언어, 튜토리얼 단위로 분리

## 서버 반영 사항

- `GET /community/users/{user_id}` 작성자 프로필 API 추가
- 게시글·댓글·답글·후기 응답에 `updated_at`, `author_is_admin` 추가
- 알림 응답에 댓글·답글 대상 ID 추가
- 기존 SQLite DB의 수정 시각 컬럼 자동 보정 추가
