# 내가 올린 레시피 상세 조회 수정

## 원인

- `내가 올린 레시피` 목록은 `ProfileProvider`가 회사 API의 `/recipe/personal_recipes/100`을 직접 호출하여 표시합니다.
- 레시피 상세 화면은 별도의 `RecipeProvider` 메모리 목록에서 `recipeId`를 검색합니다.
- 앱 시작 직후 회사 인증 토큰이 복원되기 전에 `RecipeProvider.loadRecipes()`가 먼저 실행되면 개인 레시피 조회가 401로 실패할 수 있습니다.
- 이 경우 마이페이지 목록에는 레시피가 보이지만 상세 화면의 `RecipeProvider`에는 해당 레시피가 없어서 `레시피를 찾을 수 없습니다.`가 표시됩니다.

## 수정 내용

1. `내가 올린 레시피` 화면 진입 및 새로고침 시 `ProfileProvider`와 `RecipeProvider`를 함께 갱신합니다.
2. 목록 항목을 누를 때 해당 레시피가 메모리에 없으면 회사 레시피 목록을 다시 불러옵니다.
3. 상세 화면을 직접 열었을 때도 자동으로 한 번 재조회합니다.
4. 재조회 실패 시 즉시 `찾을 수 없습니다`만 표시하지 않고 오류 내용과 `다시 불러오기` 버튼을 표시합니다.

없습니다. Flutter 앱 코드만 변경되었습니다.
