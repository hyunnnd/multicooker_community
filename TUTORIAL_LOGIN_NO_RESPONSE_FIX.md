# 신규 회원 로그인 무응답 수정

- 로그인 성공 후 `/users/me/settings` 응답을 최대 15초 이상 기다리던 흐름을 제거했습니다.
- 개인 API `/auth/me` 응답에 `tutorial_completed`를 포함해 기존 로그인 과정에서 바로 신규 여부를 판별합니다.
- 신규 계정은 즉시 `/tutorial/home`으로 이동합니다.
- 구버전 서버 호환을 위한 설정 조회는 3초 제한을 두고 실패해도 홈으로 이동합니다.
- 튜토리얼 확인 중에는 로그인 로딩 표시가 유지됩니다.
- DB 파일과 기존 사용자 데이터는 변경하거나 포함하지 않습니다.
