# 마이페이지 디자인 수정 내역

수정 파일: `lib/features/profile/presentation/profile_pages.dart`

## 적용 화면

- 마이페이지 > 레시피 관리
- 레시피 관리 > 내가 올린 레시피
- 레시피 관리 > 저장한 레시피
- 마이페이지 > 내가 쓴 후기
- 후기 수정 및 삭제 확인 팝업

## 주요 변경 사항

- 배경 `#F8FAFC`, 카드 흰색, 테두리 `#E5E7EB`, 카드 모서리 16px 적용
- 주요 동작 색상 Graphene Orange `#F97316` 적용
- 카드 그림자 제거 후 1px 테두리 중심으로 통일
- 좌우 여백 16px, 카드 간격 12px 적용
- 레시피 관리 상단을 요약 카드와 탭 버튼 구조로 변경
- 레시피 카드에 썸네일, 공개 상태, 온도, 시간, 작성자 정보를 계층적으로 배치
- 내가 쓴 후기 상단에 후기 개수와 평균 별점 요약 카드 추가
- 후기 내용 영역, 레시피 이동 버튼, 수정·삭제 메뉴를 카드형 UI로 정리
- 로딩, 빈 화면, 오류 안내, 확인 팝업 스타일 통일

## 확인 사항

현재 작업 환경에 Flutter SDK가 없어 `flutter analyze`와 실제 기기 렌더링 검증은 수행하지 못했습니다. Dart 구문 괄호 균형과 변경 파일 구조는 별도로 확인했습니다.
