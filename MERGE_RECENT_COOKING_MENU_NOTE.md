# 최근 조리·더보기 메뉴 UI 병합

- 커뮤니티 검색창의 힌트/입력 텍스트를 세로 중앙 정렬
- 업로드된 appmaster (11)(2)의 최근 조리 카드 디자인 적용
- 최근 조리의 온도, 시간, 완료 상태 정보를 카드 하단에 표시
- 최근 조리 더보기 메뉴에서 같은 설정으로 조리, 레시피로 저장, 이력 삭제 제공
- 커뮤니티와 마이페이지의 모든 더보기 버튼을 공통 회색 둥근 버튼 디자인으로 통일
- 공통 팝업 메뉴의 흰색 배경, 테두리, 모서리 및 그림자 스타일 통일
- DB, .env, local_uploads, logs, build 결과물은 포함하지 않음
