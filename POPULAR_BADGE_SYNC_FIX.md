# 인기글 배지 동기화 수정

- 기존 문제: 인기 탭은 활동 점수가 1점 이상이면 포함했지만, `인기` 배지는 서버의 임계값 충족 여부(`is_popular`)로 표시했습니다.
- 결과: 인기 탭에 올라간 게시글 옆에 `인기` 배지가 없는 경우가 발생했습니다.
- 수정: 인기 탭 포함 기준과 배지 표시 기준을 모두 서버의 `is_popular` 값으로 통일했습니다.
- 최근 3일 활동 점수가 `POPULAR_SCORE_THRESHOLD` 이상이거나 관리자가 강제 인기 설정한 게시글만 인기글로 처리됩니다.
- 좋아요 응답에 포함된 갱신된 게시글을 Provider가 교체하므로, 임계값에 도달하는 순간 목록과 상세 화면에 배지가 바로 표시됩니다.
- DB 파일, 업로드 파일, 로그, `.env`는 배포 ZIP에 포함하지 않습니다.
