# 커뮤니티 상세 화면 디자인 수정

- 커뮤니티 목록의 고정 공지 배너는 이전 정렬과 표시 방식으로 복구했습니다.
- 공지 상세 화면에서 실제 공지 제목과 날짜를 오른쪽 정렬했습니다.
- 공지 상세 화면의 바깥 배경은 #F8FAFC, 본문과 목록은 흰색 16px 카드와 1px 테두리로 구성했습니다.
- 게시글 상세 화면도 회색 바깥 배경, 흰색 카드, 16px 모서리, #E5E7EB 테두리로 통일했습니다.
- 게시글 본문, 댓글 정렬 영역, 댓글을 각각 카드 구조로 분리했습니다.
- 게시글/공지 상세 상단에는 홈 화면과 유사한 왼쪽 정렬 제목과 회색 버튼형 뒤로가기/메뉴 영역을 적용했습니다.
