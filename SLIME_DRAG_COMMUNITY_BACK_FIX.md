# 슬라임 드래그 및 커뮤니티 뒤로가기 수정

## 슬라임 드래그
- 기존에는 드래그 좌표에서 고정값 33px을 빼 전체 위젯 위치를 계산했습니다.
- 슬라임 위젯의 실제 폭은 136px이고 상태 말풍선이 표시되면 높이도 달라져, 처음 드래그할 때 오른쪽·아래로 위치가 튀었습니다.
- 드래그 시작 순간의 포인터 위치와 슬라임 위젯의 실제 좌상단 위치·크기를 저장하고, 이후 이동량만 더하도록 변경했습니다.
- 따라서 슬라임의 어느 부분을 잡아도 최초 위치를 유지한 상태로 자연스럽게 이동합니다.

## 커뮤니티 뒤로가기
- 공통 `AppBackButton`에 사용자 지정 콜백을 받을 수 있는 `onPressed`를 추가했습니다.
- 커뮤니티 게시글 상세, 작성자 프로필, 관리자, 공지사항, 게시글/후기 작성 및 검색 화면의 뒤로가기 버튼을 같은 공통 위젯으로 통일했습니다.
- 레시피·설정 화면과 동일한 36x36 회색 둥근 버튼, 회색 화살표 스타일을 사용하면서 기존 커뮤니티 내부 화면 전환 동작은 유지합니다.
