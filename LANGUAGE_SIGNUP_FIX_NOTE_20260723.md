# 언어·회원가입 수정 패치

기준 프로젝트: `GrapheneSquare_cooker_app-master (14).zip`

## 반영 내용

- 설정에서 English 선택 시 설정, 마이페이지 하위 화면, 기기 관리, 커뮤니티, 알림, 공지 및 관리자 화면의 고정 UI 문구를 영어로 표시
- 게시글 제목·내용, 댓글, 답글, 후기 내용 등 사용자 작성 데이터는 원문 유지
- 선택한 언어를 보안 저장소에 저장하여 앱 재실행 후에도 유지
- 회원가입 이메일 인증 제한시간을 3분에서 5분으로 변경
- `마케팅 수신 동의 (선택)` 문구 적용
- 비밀번호 형식 검증 및 구체적인 안내 추가
- 같은 비밀번호 오류로 가입 완료 버튼을 반복해서 눌러도 안내가 매번 다시 표시되도록 수정
- 가입 완료 버튼 전체 영역의 터치 범위 보강
- 화면 아래 오류 토스트가 버튼 터치를 가로채지 않도록 수정
- 기기 연결 성공 시 기기 관리 화면에서 홈으로 즉시 이동하는 기존 수정 유지

## 보존 사항

- 펫 코드 및 `tangerine_chef.webp`, `tangerine_idle.png`는 변경하지 않음
- 레시피·커뮤니티 디자인과 사용자 데이터 구조는 변경하지 않음
- 이 패치 ZIP에는 DB와 업로드 파일이 포함되어 있지 않음
