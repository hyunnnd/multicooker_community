# 뒤로가기 동작 정리

- AI 추천 결과 등 AI 하위 화면 → AI 탭(`/ai-scan`)
- 레시피 검색·분류·상세·등록·조리 하위 화면 → 레시피 탭(`/recipes`)
- 마이페이지 세부 화면 → 마이 탭(`/settings`)
- 쿠커·독립 조리 화면 → 실제 이전 화면이 있으면 이전 화면, 없으면 홈(`/home`)
- AI·레시피·커뮤니티·마이 탭 최상위 화면 → 홈(`/home`)
- 홈 → 2초 안에 뒤로가기를 두 번 누르면 앱 종료

`push()`로 진입한 화면은 실제 이전 화면을 우선 복원하며, `go()`로 이동해 탐색 기록이 없는 경우에는 위의 탭별 기준 경로를 사용합니다.
