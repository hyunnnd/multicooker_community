# 커뮤니티 시간·댓글 입력·저장 기능 수정

## 변경 내용

1. 게시글, 댓글, 답글, 알림의 `created_at`을 기준으로 상대 시간을 계산합니다.
   - 1분 미만: `방금 전`
   - 1시간 미만: `N분 전`
   - 24시간 미만: `N시간 전`
   - 1일: `어제`
   - 7일 미만: `N일 전`
   - 이후: `YYYY.MM.DD`
   - 화면은 30초마다 다시 계산되어 앱을 열어 둔 상태에서도 시간이 갱신됩니다.

2. 댓글·답글 입력 및 수정 TextField의 배경을 명시적으로 회색으로 고정했습니다.
   - 전역 입력 테마의 흰색 배경이 회색 컨테이너 위에 겹쳐 보이던 현상을 방지합니다.

3. 커뮤니티 게시글 저장(북마크) 기능을 제거했습니다.
   - 목록 및 상세 화면의 저장 아이콘 제거
   - Flutter Repository/Provider/Model의 북마크 로직 제거
   - FastAPI의 게시글 북마크 등록·해제 API 제거
   - 레시피 저장 기능과 관련 API는 유지됩니다.

## 서버 적용

`main.py`가 변경되었으므로 로컬 FastAPI 서버의 파일을 교체한 뒤 서버를 재시작해야 합니다.
기존 SQLite DB에 남아 있는 과거 고정 시간 문자열은 서버 시작 시 한 번만 실제 시각으로 변환됩니다.
